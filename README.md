# Power-BI-Project
This project is regarding Uber Car Analysis
A Power BI dashboard focused on analyzing Uber trip data (visualizations, trends, hotspots).

Includes .pbix report file, dataset, and documentation.

Shows how to handle typical BI tasks in Power BI: ETL with Power Query, DAX calculations, mapping, interactive visuals, and pattern analytics.

Features insights such as trip durations, pickup/drop-off hot spots, time-based booking trends, etc.

Good starting point if you want to see a real working Uber dashboard and the full Power BI project code/data.
